# Hello Word

A minimal static example webapp built with plain HTML, CSS, and JS.

## Run locally

Serve the directory with any static file server:

```bash
# Python
python3 -m http.server 8000

# Node (npx, no install needed)
npx serve .
```

Then open [http://localhost:8000](http://localhost:8000).

## Deploy to Vercel

This project is ready to deploy to Vercel as-is — no build step required. Push to a Git repository and import it in the Vercel dashboard, or use the CLI:

```bash
npx vercel
```
